🖥️ Установка Proger Build (bspwm)

Выберите подходящий вариант установки Arch Linux
🔒 Полное шифрование диска

С шифрованием

Классическая схема с LUKS + LVM. Раздел root находится внутри зашифрованного контейнера, требуется ввод пароля при загрузке.

✔ Максимальная безопасность
✔ Подходит для ноутбуков и рабочих станций

👉 Инструкция с шифрованием
⚡ Простая схема UEFI/GPT

Без шифрования

Обычная установка с двумя разделами (EFI + root). Никаких дополнительных паролей, быстрая загрузка.

✔ Простота и скорость
✔ Подходит для виртуальных машин и тестовых систем

👉 Инструкция без шифрования
🔗 Исходный репозиторий конфигурации: Zproger/bspwm-dotfiles
