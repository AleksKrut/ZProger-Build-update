Установка Proger Build (bspwm) на Arch Linux
Без шифрования (простая схема UEFI/GPT)

Оригинальное видео: YouTube, репозиторий: GitHub.

1. Подготовка и SSH

Установите openssh:

pacman -Sy --noconfirm openssh

Узнайте IP:

ip address

Задайте пароль root (если ещё не задан):

passwd

2. Установка micro и пакетов

pacman -Sy micro
pacman -S dhcpcd
pacman -S iwd

Очистка экрана:

clear

3. Разметка диска (UEFI GPT)

Просмотр дисков:

lsblk

Разметка (пример для /dev/sda):

parted /dev/sda
mklabel gpt
mkpart ESP fat32 1MiB 512MiB
set 1 boot on
mkpart primary ext4 512MiB 100%
quit

4. Форматирование и монтирование

mkfs.ext4 /dev/sda2
mkfs.fat -F 32 /dev/sda1
mount /dev/sda2 /mnt
mkdir /mnt/boot
mount /dev/sda1 /mnt/boot

5. Установка базовой системы

pacstrap -K /mnt base linux linux-firmware base-devel dhcpcd net-tools iproute2 networkmanager vim micro efibootmgr iwd
genfstab -U /mnt >> /mnt/etc/fstab
arch-chroot /mnt

6. Настройка локалей, времени, пользователя

micro /etc/locale.gen
locale-gen
ln -sf /usr/share/zoneinfo/Europe/Kiev /etc/localtime
hwclock --systohc
echo "arch" > /etc/hostname
passwd
useradd -m -G wheel,users,video -s /bin/bash user
passwd user
systemctl enable dhcpcd
systemctl enable iwd.service

7. Настройка mkinitcpio (без encrypt/lvm2)

Оставьте HOOKS по умолчанию, ничего не добавляйте.

mkinitcpio -p linux

8. Установка загрузчика systemd-boot

bootctl install --path=/boot
micro /boot/loader/loader.conf

Содержимое loader.conf:

timeout 3
default arch
micro /boot/loader/entries/arch.conf

Содержимое arch.conf (UUID получите командой blkid /dev/sda2):

title Arch Linux by ZProger
linux /vmlinuz-linux
initrd /initramfs-linux.img
options root=UUID=ваш_uuid rw
sudo EDITOR=micro visudo

Раскомментируйте строку %wheel ALL=(ALL:ALL) ALL

exit
umount -R /mnt
reboot

9. После перезагрузки (подключение по SSH)

sudo systemctl enable sshd
sudo systemctl start sshd

10. Установка окружения bspwm

sudo pacman -Syu
sudo pacman -S xorg bspwm sxhkd xorg-xinit xterm git python3
micro /etc/X11/xinit/xinitrc

Добавьте в конец:

exec bspwm
git clone https://github.com/Zproger/bspwm-dotfiles.git
cd bspwm-dotfiles
python3 Builder/install.py

Запуск графической оболочки:

startx
